# OSGC Architecture

## Flujo operacional

Fuentes -> Ingestion -> Normalizacion -> Casos -> Eventos -> SLA -> Alertas -> Auditoria -> Observabilidad

## Capas

- UI operacional
- API operacional
- Dominio contractual
- Motor SLA
- Auditoria
- Persistencia

## Objetivo

Construir un observatorio institucional para seguimiento contractual y vigilancia operacional.
